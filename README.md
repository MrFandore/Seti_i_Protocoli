## [Практическая работа №3: Автоматизация сетевого оборудования с использованием Python](https://github.com/MrFandore/Seti_i_Protocoli/tree/main/Prac_3)

### Описание проекта
Проект демонстрирует автоматизацию настройки сетевого оборудования Cisco с использованием библиотеки Netmiko для Python. Скрипт выполняет подключение к роутеру и коммутатору, сбор конфигурационной информации и управление VLAN.

### Функциональность
- Установление SSH-соединения с сетевым оборудованием
- Сбор текущей конфигурации устройств
- Мониторинг состояния интерфейсов
- Управление VLAN (удаление и создание)
- Анализ ARP-таблиц и MAC-адресов

### Технические требования
- **ОС**: CentOS 9 (рекомендуется виртуальная машина)
- **Python**: 3.x
- **Необходимые пакеты**: netmiko, ipaddress, ipython

### Структура проекта
```
Prac_3/
├── main.py              # Основной скрипт автоматизации
└── Лабораторная работа 3(Метода).docx
```

### Использование
```bash
python3 main.py
```

## [Практическая работа №4: Сбор конфигурации сетевых устройств с использованием Netmiko](https://github.com/MrFandore/Seti_i_Protocoli/tree/main/Prac_4)

### Описание проекта
Проект демонстрирует автоматизированный сбор конфигураций с нескольких сетевых устройств (маршрутизаторов и коммутаторов Cisco) с помощью библиотеки Netmiko для Python. Скрипт подключается к 7 сетевым устройствам и сохраняет их текущие конфигурации в отдельные файлы.

### Функциональность
- Автоматическое подключение к нескольким сетевым устройствам
- Сбор текущей конфигурации (show running-config) с каждого устройства
- Сохранение конфигураций в текстовые файлы с именами устройств
- Организация файлов в структурированную папку

### Технические требования
- **ОС**: CentOS 9 или любая другая с поддержкой Python
- **Python**: 3.x
- **Необходимые пакеты**: netmiko

### Сетевое оборудование
- R1: 192.168.181.10
- R10: 192.168.181.20
- SW1: 192.168.181.30
- SW2: 192.168.181.31
- SW10: 192.168.181.50
- SW11: 192.168.181.51
- PROVIDER: 192.168.181.70

### Структура проекта
```
Prac_4/
├── main.py                    # Основной скрипт сбора конфигураций
├── cfgs/                      # Папка для сохранения конфигураций
│   ├── R1_cfg.txt            # Конфигурация маршрутизатора R1
│   ├── R10_cfg.txt           # Конфигурация маршрутизатора R10
│   ├── SW1_cfg.txt           # Конфигурация коммутатора SW1
│   ├── SW2_cfg.txt           # Конфигурация коммутатора SW2
│   ├── SW10_cfg.txt          # Конфигурация коммутатора SW10
│   ├── SW11_cfg.txt          # Конфигурация коммутатора SW11
│   └── PROVIDER_cfg.txt      # Конфигурация маршрутизатора PROVIDER
└── Лабораторная работа 4(Тода).docx
```

## Установка и настройка

### Общие требования
```bash
# Проверка версии Python
python3 --version

# Установка менеджера пакетов
pip3 install -U pip

# Установка зависимостей
pip install netmiko
```

### Настройка сетевого оборудования
Убедитесь, что на устройствах:
- Включен SSH-доступ
- Настроены учетные данные:
    - Логин: admin
    - Пароль: cisco
    - Enable-пароль: cisco
